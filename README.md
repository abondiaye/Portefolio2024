# Portfolio - N'DIAYE Abdoulaye

Portfolio personnel présentant mon profil de **Technicien Support IT N1-N2** (9 ans d'expérience) et ma double compétence en **développement web Fullstack**.

🔗 **Voir le site en ligne :** https://abondiaye.github.io/Portefolio2024/

## Contenu du site

- **Home** : présentation, compétences (langages, frameworks, outils)
- **Projets** : réalisations (portfolio, projets de BTS)
- **Parcours** : formations et expériences professionnelles
- **Hobbies** : centres d'intérêt et bénévolat
- **Contact** : formulaire de contact / demande de CV

## Stack technique

- HTML / CSS / JavaScript vanilla
- [Swiper](https://swiperjs.com/) pour les carrousels d'images
- [particles.js](https://vincentgarreau.com/particles.js/) pour le fond animé
- PHP pour le traitement du formulaire de contact (hors ligne sur GitHub Pages, hébergement statique uniquement)

## Lancer le site en local

```bash
php -S localhost:8888
```

Puis ouvrir http://localhost:8888/index.html dans le navigateur.

Tâches effectuées

Site conçu avec le framework Symfony sous forme d'API. L'idée est de permettre d'ajouter des images temporaires pendant le développement d'un site. Il suffit d'appeler l'url dans la source d'une balise image pour avoir une image aléatoire suivant la catégorie choisie.

Stack

API d'images aléatoires

L'idée est venue en développant des sites et en cherchant des images de substitutions en attendant les vrais assets.

Pour ce faire j'ai utilisé Symfony. En tant qu'administrateur on peut mettre en base de données des images avec un tag pour l'identifier. Ensuite les utilisateurs ont juste à mettre la route API dans leur balise img pour récupérer une image souhaitée.

En utilisant la route www.api_images/photo/animal, l'API renverra aléatoirement une image parmi celles qui ont le tag animal. Le navigateur renverra ensuite cette même image pour les autres appels à cette même url.

Pour avoir des images différentes il faudra alors appeler la route www.api_images/photo/animal/{id}, qui renverra une image unique et qui permettra d'avoir de différentes images sur le site en développement.
